# MLOps Project

A small MLOps starter project for a rental price prediction pipeline.

## Structure

```text
data/
Dockerfile
main.py
rental_price_prediction_pipeline.yaml
requirements.txt
```

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
